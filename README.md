# 基金板块实时雷达

一个本地运行的基金板块数据系统，用于查看国内主要指数、行业/概念板块、ETF/场内基金、自选基金估值，并基于规则生成未来 2-3 个月入场节奏和已有仓位操作建议。

## 运行

```powershell
node server.js
```

然后打开：

```text
http://localhost:3100
```

## 数据口径

- 主要指数、行业板块、概念板块、ETF/场内基金：东方财富行情中心。
- 自选基金盘中估值：天天基金估值接口；若取不到估值，则尝试东方财富场内行情。
- 历史验证：天天基金/东方财富基金页面中的 `Data_netWorthTrend` 日频净值序列，用代表ETF/基金验证当前板块轮动框架。
- 若东方财富主行情域临时 502，系统会自动切换备用行情域；单个数据源失败不会拖垮整个页面。
- “基金板块”会把行业、概念和ETF映射到更接近养基宝语境的主题分类，例如半导体/芯片、AI算力/通信、消费白酒、医药医疗、红利低波、港美海外、债券货币等。
- 宏观与政策背景：国家统计局 2026 年 5 月 PMI 解读、证监会指数化投资与公募基金费用规则公开信息。

参考入口：

- 东方财富行情中心：https://quote.eastmoney.com/center/list.html
- 天天基金：https://fund.eastmoney.com/default.html
- 国家统计局 2026 年 5 月 PMI 解读：https://www.stats.gov.cn/sj/sjjd/202605/t20260531_1963825.html
- 证监会指数化投资行动方案：https://www.csrc.gov.cn/csrc/c100028/c7536076/content.shtml
- 证监会公募基金销售费用规定：https://www.csrc.gov.cn/csrc/c100028/c7606047/content.shtml

## 策略逻辑

系统不是预测器，而是把行情信号转为纪律：

- 市场评分来自主要指数涨跌、板块上涨广度、基金板块强势占比、主力资金流入占比、短线过热惩罚和多周期信号置信度。
- 板块评分已从单日热度升级为多周期风险调整信号：综合当日、5日、10日、20日、60日、年初至今涨跌，叠加成交额、主力资金、趋势一致性、波动/振幅惩罚，并对极端值做压缩，避免单日暴涨直接满分。
- 每个基金板块会输出信号置信度；置信度来自多周期字段完整度、成交活跃度、资金字段可用性、覆盖板块数量和ETF验证覆盖度。
- 未来 2-3 个月入场建议采用分批法：首笔、回踩加仓、现金预留。
- 候选方向优先筛选成交活跃、涨幅为正、资金配合、ETF验证较好、且不过热的基金板块。
- 已有仓位建议会结合单只仓位占比、成本盈亏、基金名称、手动主题、基金重仓股行业/概念和当天涨跌，生成“持有、分批、止盈、减仓、停止补仓”等动作，并解释原因和观察点。
- 第二阶段新增历史验证：系统会为当前排名靠前的基金板块匹配代表ETF历史净值，按 20/60/120 日动量、60 日波动惩罚和 20 个交易日再平衡做轮动回测，等权买入前三名，并扣除单边 15bp 交易成本。
- 历史验证会展示策略收益、沪深300基准收益、超额收益、最大回撤、夏普、胜率、换手、成本拖累、样本外收益和板块样本覆盖；若样本不足、回撤过深或样本外跑输，会降低置信度并提示风险。
- 第三阶段新增组合优化：系统会把你的持仓映射到基金板块，结合实时评分、历史验证、权益暴露、单只集中度和主题集中度，生成目标权重、偏离度、单只基金调仓动作和下一步可补方向。
- 组合优化采用风险预算思路：先确定目标权益仓位，再把组合拆成核心、卫星、防守/现金和对冲桶；单只基金、单一高波动主题、海外和黄金都有上限，避免强信号导致过度集中。

## 风险提示

基金净值估算按历史持仓与指数走势推算，不能替代基金公司最终披露净值。系统只提供研究辅助，不构成保证收益、代客理财或个性化投资承诺。
